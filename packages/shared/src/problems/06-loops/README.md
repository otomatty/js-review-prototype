# Ch06. ループ

`for` / `while` / `for...of` と `break` / `continue`。 「同じことを繰り返す」 という発想を生で書く章。 高階関数 (Ch09) の前提となる。

## 含まれるステージ

- **S2** 文法定着 — 単純な `for (let i = 0; ...)` と `for...of`。
- **S3** ロジック入門 — 配列の走査・累積で値を 1 つ返す。
- **S4** アルゴリズム — 二重ループ、 累積、 `break` / `continue` の活用。
- **S5** 設計演習 — ループとデータ構造を組み合わせた処理。
