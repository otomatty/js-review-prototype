# Ch01. 変数

## 含まれるステージ

- **S1** 文法体験 — 1 文 1 構文で `const` / `let` の宣言と代入を体験する。
- **S2** 文法定着 — 既習構文と組み合わせて再代入・スコープを意識する。
- **S3** ロジック入門 — 関数引数として変数を扱い、 自力で命名する。
