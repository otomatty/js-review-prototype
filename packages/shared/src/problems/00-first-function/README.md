# 00. はじめての関数

JavaScript で **「関数を作る」「値を返す」「引数を受け取る」** という、 ほかのすべての章の前提となる最小の操作だけを扱う章。 プログラミングを初めて触る人の **最初の 5 問** として用意されています。

> この章には対応する単一の MDN ページがない（複数ページにまたがる基礎知識のため）。 各問題が参照する MDN セクションは個別に記載しています。

## この章が前提とすること

- 何もありません。 「コードってどう書くの？」 のレベルから始められます。

## この章を終えて使えるようになる構文

- `function 名前(引数1, 引数2) { ... }` で関数を宣言する
- `return 値;` で関数から値を返す
- 数値リテラル ( `42`, `1.5`, `-3` ) と文字列リテラル ( `'hi'` )
- 算術演算子 `+` と `*`

これだけで章 1 「変数と型」 に進めます。

## 採点ルール（共通）

すべての課題で「**Lint・AST・テストの 3 軸すべて通過 = クリア**」の二値判定です。

- **Lint**: ESLint v9 で `severity === 2` (error) が 0 件
- **AST**: `var` の使用を禁止 (`no-var` lint と AST forbidden)
- **Tests**: 全テストケース pass

## 設計メモ（オーサリング向け）

- **Stage 0**: 関数・return・基本式のみで完結する問題に限定
- 各 starter は `function 名前(...) { return 安全な値; }` の **完成形に近い形** で提供し、 学習者は **return の中身だけ書き換える**
- `let / const` をまだ知らないため、 starter 内に変数宣言は **入れない** (関数本体は 1 行式で完結)
- `if / else` も使わない (使うと章 3 の前借りになる)
